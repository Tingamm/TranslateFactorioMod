# TranslateFactorioMod-
A toolchain for translating Factorio mods — extracts English texts from mod packages, translates them via DeepSeek AI with optional reference to existing translations, and packages them into ready-to-publish mods.Factorio 模组汉化工具链 —— 从模组压缩包提取英文文本，调用 DeepSeek AI 翻译，并打包为可直接发布的汉化模组。支持旧版汉化参考，保持翻译风格一致。
